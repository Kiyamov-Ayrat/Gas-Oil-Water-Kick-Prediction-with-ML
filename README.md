# Прогнозирование газонефтеводопроявлений при бурении скважин с использованием машинного обучения

## Описание проекта

Данный проект посвящён разработке системы раннего обнаружения газонефтеводопроявлений (ГНВП) на основе методов машинного обучения и данных телеметрии процесса бурения.

В ходе работы выполняются:

* загрузка и обработка сырых данных бурения;
* автоматическая разметка событий ГНВП;
* генерация информативных признаков;
* обучение моделей машинного обучения;
* оценка качества моделей;
* визуализация результатов и анализ важности признаков.

Цель проекта - повысить безопасность буровых работ за счёт раннего обнаружения признаков поступления пластового флюида в ствол скважины.

## Используемые модели

В проекте реализовано сравнение трёх алгоритмов машинного обучения:

### Random Forest

Ансамблевый алгоритм на основе множества деревьев решений.

### XGBoost

Градиентный бустинг над деревьями решений с высокой производительностью и устойчивостью к шуму.

### CatBoost

Алгоритм градиентного бустинга от Яндекса, хорошо работающий на сложных табличных данных.

## Формирование признаков

Для повышения качества прогнозирования были сформированы дополнительные признаки:

### Скользящие статистики

Для временных окон:

* 30 секунд
* 300 секунд
* 600 секунд

Рассчитывались:

* среднее значение;
* стандартное отклонение;
* максимальное значение.

### Производные признаки

* изменение объёма PVT за 10 секунд;
* изменение объёма PVT за 60 секунд;
* изменение давления SPP;
* изменение объёма бурового раствора;
* разница между PVT Total и PVT Monitor;
* разность между глубиной скважины и глубиной долота.

### Временные признаки

* час суток;
* день недели.

## Результаты

Сравнение моделей на тестовой выборке:

| Модель        | ROC-AUC    | PR-AUC     | Recall   | Precision | False Positive | False Negative |
| ------------- | ---------- | ---------- | -------- | --------- | -------------- | -------------- |
| Random Forest | 0.9470     | 0.7327     | 0.91     | 0.12      | 42755          | 581            |
| XGBoost       | 0.9408     | 0.7278     | 0.90     | 0.09      | 60932          | 647            |
| CatBoost      | **0.9610** | **0.7580** | **0.92** | 0.07      | 76974          | **526**        |

### Выводы

Наилучшее качество показала модель CatBoost, продемонстрировав:

* максимальное значение ROC-AUC;
* максимальное значение PR-AUC;
* наибольшую полноту (Recall);
* минимальное количество пропущенных событий ГНВП (False Negative).

При этом все модели демонстрируют низкое значение Precision, что приводит к большому числу ложных срабатываний. Это связано с сильным дисбалансом классов и приоритетной оптимизацией Recall, что является ожидаемым поведением для задач раннего обнаружения аварийных ситуаций.

Для задач предупреждения аварий при бурении пропуск события является наиболее критичной ошибкой, поэтому CatBoost был признан наиболее эффективной моделью в рамках данного исследования.
